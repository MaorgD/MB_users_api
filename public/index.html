Toys Guide

With these "GET" requests you can get information about toys. By adding these queries you can edit the request :
  • perPage - selecting the amount of objects on the page.
  • page - selecting the amount of pages.
  • sort- selecting sorting subject (price,category,id,name).
  • reverse - change the sort dirction.
By login you can add, edit and remove your own toys, admin can delete and edit all users the toy.
